The least path we can construct between $x$ and $y$ is $(x)$ in the case of $x = y$. So $d(x,y) \geq 0$.

if $x = y$, then $(x)$ is a valid path, concluding $d(x,y) = 0$. If $d(x,y) = 0$ then the graph contains a path $(x) = (y)$, concluding $x = y$.

Since the graph is undirected, It is clear $(v, v_1, \dots, v_k)$ is the least $vv_k$ path if and only if $(v_k, v_{k-1}, \dots, v)$ is the least $v_kv$ path. It follows $d(x,y) = d(y,x)$.

The paths $(x, \dots, y)$ and $(y, \dots, z)$ can be concatenated to construct a new path $(x, \dots, z)$ whose length is $d(x,y) + d(y,z)$. If we considered the minimal-length $xz$ path, Then by definition we get $d(x,z) \leq d(x,y) + d(y,z)$. $\blacksquare$.

Fact. A connected tree of order $n$ has exactly $n-1$ edges.

Let $n_i$ denote the order of $i$th connected tree. Then by the Fact $i$th tree has $n_i - 1$ edges. It follows the total number of edges of the forest is $\sum_{i=1}^k (n_i - 1) = n - k$.
